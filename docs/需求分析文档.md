## 一、产品目标

### 1.1 核心价值

智慧城市平台面向建筑能耗管理场景，通过多层级可视化视图（园区 → 建筑 → 楼层 → 区域），帮助管理人员：

- **实时掌握**建筑各楼层能耗状态
- **直观对比**当前用量与 AI 优化后预期用量
- **快速响应**AI 建议，一键应用节能策略
- **追踪月度节省**效果（费用 / 能源 / 碳排放）

### 1.2 设计原则

| 原则 | 描述 |
|------|------|
| 信息密度适中 | 左侧面板承载数据，右侧视图承载空间感知 |
| 渐进式呈现 | 动画逐步引导用户注意力，避免信息轰炸 |
| 空间与数据联动 | 楼层视图点击 = 数据聚焦，保持上下文一致 |
| 深色沉浸风格 | 暗色背景 + 蓝绿荧光，符合智慧城市科技感 |
