### 1.2 关键架构模式

| 模式 | 说明 |
|------|------|
| `EProvider` / `useE()` | 全局 Context：floors / active / tab / range / setRange / ecTop / setEcTop / introReady / setIntroReady |
| `RANGE_DATA` | 单一数据常量（Day/Month/Year），所有图表和卡片的唯一数据来源 |
| `panelKey` 重挂载 | Refresh 或 Range 切换时 `setPanelKey(k+1)`，触发所有图表重新 mount 并重播入场动画 |
| `introReady` 门控 | EnergyChart / SavingsChart 仅在 Intro 关闭后才初始化，确保动画与交互同步 |
| `ResizeObserver` 对齐 | 测量 EnergyChart 容器 `offsetTop` → 存入 `ecTop` → 右侧面板 `paddingTop` 动态跟随 |
| `lineDrawPlugin` | Chart.js 自定义插件，RAF 动画循环驱动折线从左到右绘制 |
| `AnimVal` 组件 | 封装 RAF 数字 count-up，支持整数和小数，用于右侧面板卡片 |
