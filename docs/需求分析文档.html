一、产品目标

1.1 核心价值

智慧城市平台面向建筑能耗管理场景，通过多层级可视化视图（园区 → 建筑 → 楼层 → 区域），帮助管理人员：

    • 实时掌握建筑各楼层能耗状态

    • 直观对比当前用量与 AI 优化后预期用量

    • 快速响应AI 建议，一键应用节能策略

    • 追踪月度节省效果（费用 / 能源 / 碳排放）

1.2 设计原则

原则描述
信息密度适中左侧面板承载数据，右侧视图承载空间感知
渐进式呈现动画逐步引导用户注意力，避免信息轰炸
空间与数据联动楼层视图点击 = 数据聚焦，保持上下文一致
深色沉浸风格暗色背景 + 蓝绿荧光，符合智慧城市科技感

二、用户角色

角色描述核心需求
能源管理员负责整栋楼能耗管理查看整体用量、应用 AI 优化建议
楼层负责人负责某层楼的运营查看本层用量、对比优化前后数据
系统演示员向客户展示平台能力流畅动画、直观可视化、完整演示流程
